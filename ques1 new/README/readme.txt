Project README
Overview
This project is a web application for managing friend data, including features such as adding new friends, editing existing friend details, deleting friends, and viewing friend information. The application is built using HTML, CSS, and JavaScript for the client-side functionality, and it communicates with a server-side API to perform CRUD (Create, Read, Update, Delete) operations on friend data.

Usage Instructions
Installation
Clone the repository to your local machine.
Navigate to the project directory in your terminal or command prompt.
Install dependencies by running npm install.
Running the Application
Start the development server by running npm start.
Open your web browser and navigate to http://localhost:3000 to access the application.

Error Handling: Basic error handling is implemented, but for production use, more robust error handling and validation should be added, especially on the server side.

Testing Environment: The project assumes a local development environment. Adjustments may be necessary for deployment to a production environment, such as configuring environment variables.
